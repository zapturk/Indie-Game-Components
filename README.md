# Indie Game Components

A collection of high-quality, reusable components for Godot 4.6.

## Features

- **Component-Based Architecture**: Modular and easy to integrate.
- **Godot 4.6 Ready**: Leverages the latest engine features.
- **Easy Installation**: Available via the Godot Asset Library.

## Installation

### Via Godot Asset Library
1. Open Godot Engine.
2. Go to the **AssetLib** tab.
3. Search for "Indie Game Components".
4. Click **Download** and then **Install**.

### Manual Installation
1. Clone this repository into your project's root folder.
2. Move the `addons/indie_game_components` folder into your project's `addons` directory.
